= Médecine

== Les différents domaines

Les différents domaines de la santé sont les suivants :

* la santé publique ;
* l'épidémiologie ;
* la médecine ;
* la pharmacologie ;
* la psychologie ;
* la psychiatrie ;
* la psychanalyse ;
* l'anatomie ;
* la physiologie.
